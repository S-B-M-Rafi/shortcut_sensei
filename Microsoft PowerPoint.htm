PowerPoint Presentation Shortcuts

Even if you're quite familiar with Microsoft PowerPoint, you might be taken aback by the range and diversity of keyboard shortcuts you can use to accelerate your tasks and generally enhance convenience.

Now, does anyone anticipate you to commit all these keyboard combinations to memory? Certainly not! Each person's requirements differ, so some shortcuts will prove more beneficial to you than others. Even acquiring a handful of new techniques can be valuable. We've also aimed to keep the list neat and straightforward, so feel free to print it out if that aids you!

Furthermore, although our compilation of shortcuts here is fairly extensive, it's not an exhaustive roster of every keyboard combo accessible in PowerPoint. We've focused on the more universally practical shortcuts. Additionally, you'll be pleased to learn that nearly all of these shortcuts have been around for quite some time, thus they should prove beneficial regardless of which version of PowerPoint you're utilizing.

Note: Present keyboard combinations using the following convention. A plus (+) signifies that you should press those keys simultaneously. A comma (,) signifies that you should press keys sequentially. For instance, "Ctrl+N" indicates holding down the Ctrl key while pressing the N key and then releasing both keys. Conversely, "Alt+N,P" entails holding down the Alt key, pressing the N key, releasing the N key, pressing the P key, and then releasing all keys.

Basic Program Commands

Let's start by exploring fundamental keyboard shortcuts for initiating, terminating, and switching between presentations, along with navigating the Ribbon interface.

Shortcut Description
Ctrl+N Create a new presentation
Ctrl+O Open an existing presentation
Ctrl+S Save a presentation
F12 or Alt+F2 Open the Save As dialog box
Ctrl+W or Ctrl+F4 Close a presentation
Ctrl+Q Save and close a presentation
Ctrl+Z Undo an action
Ctrl+Y Redo an action
Ctrl+F2 Print Preview View
F1 Open the Help pane
Alt+Q Go to the “Tell me what you want to do” box
F7 Check spelling
Alt or F10 Turn key tips on or off
Ctrl+F1 Show or hide the ribbon
Ctrl+F Search in a presentation or use Find and Replace
Alt+F Open the File tab menu
Alt+H Go to the Home tab
Alt+N Open the Insert tab
Alt+G Open the Design tab
Alt+K Go to the Transitions tab
Alt+A Go to the Animations tab
Alt+S Go to the Slide Show tab
Alt+R Go to the Review tab
Alt+W Go to the View tab
Alt+X Go to the Add-ins tab
Alt+Y Go to the Help tab
Ctrl+Tab Switch between open presentations

Navigating Text, Objects, and Slides

Utilize keyboard shortcuts to swiftly move through your presentation. Experiment with these shortcuts to efficiently select text within text boxes, objects on slides, or slides in your presentation.

Shortcut Description
Ctrl+A Select all text in a text box, all objects on a slide, or all slides in a presentation (for the latter, click on a slide thumbnail first)
Tab Select or move to the next object on a slide
Shift+Tab Select or move to the previous object on a slide
Home Go to the first slide, or from within a text box, go to the beginning of the line
End Go to the last slide, or from within a text box, go to the end of the line
PgDn Go to the next slide
PgUp Go the previous slide
Ctrl+Up/Down Arrow Move a slide up or down in your presentation (click on a slide thumbnail first)
Ctrl+Shift+Up/Down Arrow Move a slide to the beginning or end of your presentation (click on a slide thumbnail first)

Efficient Formatting and Editings

Master these keyboard shortcuts to streamline your editing and formatting tasks effortlessly!

Shortcut Description
Ctrl+X Cut selected text, selected object(s), or selected slide(s)
Ctrl+C or Ctrl+Insert Copy selected text, selected object(s), or selected slide(s)
Ctrl+V or Shift+Insert Paste selected text, selected object(s), or selected slide(s)
Ctrl+Alt+V Open the Paste Special dialog box
Delete Remove selected text, selected object(s), or selected slide(s)
Ctrl+B Add or remove bold to selected text
Ctrl+I Add or remove italics to selected text
Ctrl+U Add or remove underline to selected text
Ctrl+E Center a paragraph
Ctrl+J Justify a paragraph
Ctrl+L Left align a paragraph
Ctrl+R Right align a paragraph
Ctrl+T Open the Font dialog box when text or object is selected
Alt+W,Q Open the Zoom dialog box to change the zoom for the slide
Alt+N,P Insert a picture
Alt+H,S,H Insert a shape
Alt+H,L Select a slide layout
Ctrl+K Insert a hyperlink
Ctrl+M Insert a new slide
Ctrl+D Duplicate the selected object or slide (for the latter, click on a slide thumbnail first)

Reviewing and Editing

When it's time to begin your slideshow, these keyboard combinations will be quite helpful.

Shortcut Description
F5 Start the presentation from the beginning
Shift+F5 Start the presentation from the current slide (useful for testing individual slides)
Ctrl+P Annotate with the Pen tool during a slideshow
N or Page Down Advance to the next slide during a slideshow
P or Page Up Return to the previous slide during a slideshow
B Change the screen to black during a slideshow; press B again to return to the slideshow
Esc End the slideshow

The more you incorporate keyboard shortcuts, the more they become ingrained in your memory. No one expects you to memorize all of them, but hopefully, you've discovered a few new ones that can enhance your experience with Excel and make your work a bit smoother.
